# 📊 Loan Repayment Analysis Dashboard (Power BI)

## 📌 Overview

This project presents a comprehensive analysis of loan repayment behavior using Power BI.
The dashboard focuses on identifying repayment patterns, risk exposure, and key factors such as gender, loan term, and age group.

---

## 🎯 Objectives

* Analyze loan repayment performance across different segments
* Identify high-risk groups based on demographics and loan duration
* Compare repayment behavior between male and female borrowers
* Provide actionable insights to improve credit risk management

## 📂 Dataset Description

The dataset includes the following key fields:

* **Loan Status** (PaidOff, Collection, Collection PaidOff)
* **Principal Amount**
* **Loan Terms** (7, 15, 30 days)
* **Borrower Age**
* **Education Level**
* **Gender**
* **Past Due Days**

---

## 📊 Key Insights

### 🔹 Gender Analysis

* Male borrowers account for **84.56% (399K)** of total loans
* Female borrowers represent **15.44% (73K)**
* Males dominate repayment volume but also carry higher absolute risk
* Females show relatively more stable repayment behavior

---

### 🔹 Loan Term Analysis

* **7-day loans** have the highest on-time repayment rate
* **Longer terms (30 days)** show increased risk and collection dependency
* Clear trend: **longer duration → higher default risk**

---

### 🔹 Age Group Analysis

* **25–35** age group drives the highest loan volume and repayments
* **18–24** group shows relatively higher risk proportion
* **46–60** contributes the least to overall portfolio

---

## 📈 Features Implemented

* Interactive slicers (Loan Status, View Type)
* Dynamic measure switching (Actual vs Percentage)
* Age group segmentation using DAX
* Comparative visuals (Donut, Bar, Stacked Charts)

---

## 🧠 Key Takeaways

* High-volume segments (male, age 25–35) drive overall performance but require risk monitoring
* Younger borrowers and longer loan terms show higher default tendencies
* Data-driven segmentation can significantly improve lending strategies

---

## 🚀 Recommendations

* Apply stricter credit checks for younger borrowers (18–24)
* Limit exposure to longer-term loans (30 days)
* Monitor high-volume segments for early risk detection
